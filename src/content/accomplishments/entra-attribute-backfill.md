When I joined, Entra ID held little more than names and email addresses — no department, manager, hire date, or employee type — which blocked most identity-driven automation. Led a company-wide backfill:

- Defined the attribute set required for automation, reporting, and governance
- Partnered with leadership on a structured data collection process via standardized templates
- Backfilled attributes across nearly all active accounts, including external partner users operating in the tenant
- Established expectations for maintaining accurate identity data going forward

Result: a complete, automation-ready directory that directly enabled dynamic distribution lists, automated security groups, org charts (including exports for banking requirements), and CodeTwo signature automation.
