Drove a company-wide initiative to standardize Entra ID user attributes (department, manager, hire date, employee type) across the tenant, building the identity foundation needed for downstream automation:

- Defined the attribute set required for automation, reporting, and governance
- Partnered with leadership on a structured data collection process using standardized templates
- Standardized attributes across nearly all active accounts, including external partner users operating in the tenant
- Established expectations for maintaining accurate identity data going forward

Result: a complete, automation-ready directory that directly enabled dynamic distribution lists, automated security groups, org charts (including exports for banking requirements), and CodeTwo signature automation.
