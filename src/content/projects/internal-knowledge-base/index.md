## Overview

Built and deployed an internal self-service platform to address onboarding friction and fragmented knowledge management across a growing, distributed organization.

## What Was Done

- Architected and deployed the platform on Azure with SSO integration for secure, seamless access
- Organized and migrated critical documentation from scattered sources into a structured, searchable system
- Built self-service resources for common IT requests, reducing ticket volume
- Established governance standards for content ownership and maintenance

## Outcome

Scaled onboarding efficiency and reduced dependency on IT for routine information requests. Improved knowledge access across departments and created a foundation for ongoing documentation standards.

## Stack

Azure · SSO / Microsoft Entra ID · SharePoint · M365
