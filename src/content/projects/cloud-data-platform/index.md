## Overview

Designed and delivered end-to-end cloud data platforms for multiple clients as an independent consultant, enabling real-time analytics and improving operational visibility.

## What Was Done

- Built cloud data warehouses on Snowflake and Databricks with automated ingestion pipelines
- Developed API integrations to streamline vendor and client data exchange
- Implemented data quality monitoring with Metaplane and orchestrated pipelines via Apache Airflow and Kestra
- Designed compliant, scalable data systems in partnership with business stakeholders
- Deployed transformation layers using DBT and Coalesce for analytics-ready data models

## Outcome

Delivered real-time analytics capabilities and improved reporting accuracy, enabling faster and more confident decision-making across client organizations.

## Stack

Snowflake · Databricks · DBT · Coalesce · Fivetran · Apache Airflow · Kestra · Matillion · Python · AWS · Azure · GCP
